jQuery.msgbox

jQuery.msgbox is a lightweight plugin that simplifies using jQuery UI dialogs by creating a #msgbox div, and using some basic default options, effectively minifying the amount of HTML and scripting necessary to load a simple jQuery UI dialog box.

Click on the icon to preview the dialog boxes generated from the examples.

Basic Syntax

$.msgbox "Dialog Title", "Lorem ipsum dolor sit amet."

Custom Buttons

$.msgbox "Title", "Nunc sed augue dui. Aenean porttitor.", buttons:
  "Yes": ->
    alert "You pressed Yes."
    $.msgbox "close"
  "No": ->
    alert "You pressed No."
    $.msgbox "close"
